# Obstacle-Dodge
Physics-based Unity movement prototype focused on smooth acceleration, camera polish, and gameplay feedback.

A small Unity prototype focused on building a smooth, physics-based character movement system with polished camera behavior and gameplay feedback.

Features

Rigidbody-based movement

Smooth acceleration & deceleration

Diagonal speed normalization

Camera smoothing

Basic hit feedback

Screen shake system

Purpose

This project is a sandbox to experiment with core game feel principles such as responsiveness, feedback, and movement polish.
