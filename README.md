# Obstacle Dodge
A fast-paced 3D obstacle course game built in Unity 6 where you navigate through increasingly dangerous zones while dodging projectiles, falling objects, and spinning hazards.

# About
Obstacle Dodge is my first Unity game, built as a learning project to explore core game development concepts including player movement, physics, collision detection, UI systems, and C# scripting.

The goal is simple — get through all three zones without losing all your lives!

# How to Play

WASD — Move the player

Reach the end of Zone 3 to complete the course

Avoid all hazards — each hit costs a life

Reach a checkpoint to save your progress
